How to use the `add_integer` function
-------------------------------------

This module defines a function that adds 2 integers or floats (casting floats to integers).

Usage:
------

```python
>>> add_integer = __import__('0-add_integer').add_integer
>>> add_integer(2, 3)
5

# Works with floats, casting them to integers
>>> add_integer(100.5, 25.2)
125

# The second argument is optional; defaults to 98
>>> add_integer(2)
100

Handling Non-Integer Inputs:
----------------------------

If either input is not an integer or float, the function raises a TypeError.

>>> add_integer("100", 2)
TypeError: a must be an integer

>>> add_integer(2, "5")
TypeError: b must be an integer

>>> add_integer([1,2], 3)
TypeError: a must be an integer

Note: The error message "a must be an integer" or "b must be an integer" is used to indicate that inputs must be integers or floats (which are cast to integers).
